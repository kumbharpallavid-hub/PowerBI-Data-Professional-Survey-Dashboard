# 📊 Data Professional Survey Dashboard

## Project Overview

This project is an interactive Power BI dashboard built using survey data from data professionals. It provides insights into salaries, job roles, programming languages, work-life balance, and career satisfaction.

## Tools Used

- Power BI
- Power Query
- DAX
- Data Visualization

## Dashboard Features

- Country-wise Job Role Distribution
- Average Salary by Job Title
- Survey Participants Count
- Average Age Analysis
- Work-Life Balance Rating
- Happiness with Salary
- Favorite Programming Language
- Salary Analysis by Gender
- Difficulty Breaking into Data Careers
